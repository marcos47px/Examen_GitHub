Practica git – Marcos/Jesús

Git es un sistema de control de proyectos y control de versiones de código. Permite trabajar en colaboración con otras personas involucradas en el proyecto. Nos ayuda a plnificar estos proyectos y realizar un seguimiento del trabajo.

Ventajas de usar Git

  1. Descargar el proyecto y las versiones desde cualquier lugar
  2. Trabajar en grupo con mayor coordinación y profesionalidad
  3. Nos avisa cuando un compañera ha subido cualquier modificación
  4. Nos indica fallos en el código
  5. Nos indica las líneas que se ahn modificado
